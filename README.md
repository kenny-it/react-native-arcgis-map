# react-native-arcgis-map

React Native ArcGIS Mapview component for Android only

## Installation

```sh
npm install react-native-arcgis-map
```

## Usage

```js
import { ArcgisMapView } from "react-native-arcgis-map";

// ...

<ArcgisMapView color="tomato" />
```
